ToDo

Useful when developing API Gateway policies or method based ACLs. Allows you to list, create, update and delete tasks.

Methods

GET, POST, PUT, DELETE

Content type

application/json

Create task

curl -sX POST {{ template_base_url }}/tasks -H 'Content-Type: application/json' -d '{"title":"New task"}'

Get tasks

curl -sX GET {{ template_base_url }}/tasks GET /api/tasks

Update task

curl -sX POST {{ template_base_url }}/tasks/task_id -H 'Content-Type: application/json' -d '{"title":"Updated task"}'

Delete task

curl -sX DELETE {{ template_base_url }}/tasks/task_id {% endblock %}
